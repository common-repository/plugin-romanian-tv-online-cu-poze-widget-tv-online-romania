== Description ==

RO: Cu ajutorul acestui plugin puteti afisa intr-un widget posturile dumneavoastra preferate. 
    Acest plugin va permite sa alegeti dumneavoastra posturile tv pe care le afisati pe site-ul sau blogul dumneavoastra.

PS: Toate linkurile sunt NOfollow.

EN : This plugin gives you the possibility to display a widget with your favourite TV stations and allows you to choose which TV stations you want to display on your site or blog.
PS : All links are NOfollow



== Installation ==


RO:   Pasii care trebuie urmati pentru a instala pluginul:
    1. Descarcati pluginul, dezarhivati-l apoi urcati directorul tvwidget in folderul /wp-content/plugins/.
    2. Accesati partea de administrare a blogului, intrati la Plugins si activati pluginul "TVWidget".
    3. Din meniul principal accesati butonul TV Widget. Acolo veti putea administra categoriile pe care doriti sa le afisati pe blogul dumneavoastra.
    4. Daca doriti sa afisati posturile apartinand anumitor categorii, bifati categoriile respective apoi apasati butonul Listeaza Posturi TV. Din lista afisata bifati posturile pe care doriti sa le afisati in sidebarul blogului urmand apoi sa bifati Salveaza Posturi TV.
    5. In final mergeti la Appearance > Widgets si alegeti pozitia widgetului in sidebar.  

EN: Steps you should follow in order to install the plugin:

1. Download the plugin, unzip it and upload the tvwidget directory in the /wp-content/plugins/ folder. 
2. Go to the blog management console, click in Plugins and activate the TVWidget plugin. 
3. You can manage the categories you want to be displayed on your blog by going to the main menu and accessing TV Widgets. 
4. If you want to display channels from certain categories, simply check the categories you want and press the List TV Stations button. Check the channels from the list that you want to display in the blog's sidebar and then check Save TV.
5. Finally, go to Appearance > Widgets and choose the widget's position in the sidebar.
